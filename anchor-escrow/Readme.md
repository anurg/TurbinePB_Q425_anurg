##### Solana Escrow

This repo contains Solana Escrow example.

#### Clone the Repo and change directory to token-mint

```
git clone https://github.com/anurg/TurbinePB_Q425_anurg.git
cd anchor-escrow
```

##### Sync Keys

```
anchor keys sync
```

##### Build Project

```
anchor build
```

##### Run test

```
anchor test
```
